# 11. Strategic Review Board

## Objective
Decide roadmap priorities with measurable criteria.

## Decision inputs
- user value
- implementation effort
- operating cost
- policy/compliance risk
- rollback complexity

## Priority matrix
- High value + low risk: execute now
- High value + high risk: stage with guardrails
- Low value + high cost: defer

## Meeting output format
1. decision
2. reason
3. owner
4. deadline
5. rollback condition
