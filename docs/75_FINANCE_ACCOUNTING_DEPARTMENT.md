# 75. Finance and Accounting Department (Minimal)

## Mission
Keep monetization sustainable by making revenue, cost, and margin visible every week.

## Scope
- Subscription/sales income tracking
- Infra + API + tooling cost tracking
- Unit economics baseline (`revenue per active user`, `cost per active user`)
- Monthly cash discipline

## Weekly Outputs (mandatory)
1. Revenue summary (gross/net)
2. Cost summary (infra/API/tools)
3. Margin snapshot (weekly and cumulative)
4. Budget risk note (if burn rate exceeds plan)

## Guardrails
- No new paid integration without projected payback window.
- Any feature with unclear cost model stays in trial mode.
- If margin trend worsens for 2 consecutive weeks, freeze expansion work and run cost correction first.

## Handoff with Product
- Product owner provides feature usage and conversion data.
- Finance owner returns go/no-go recommendation for scaling.
