# RISK

## Security Risks
- Misconfigured MCP capability scope can over-authorize tool access.

## Privacy Risks
- Accidental reference to unapproved data sources in integrations.

## Operational Risks
- Missing required docs can break deterministic governance checks.

## Legal Risks
- Untracked external API usage can violate contractual terms.

## Approval Tier Mapping
- Governance metadata changes: medium tier.
